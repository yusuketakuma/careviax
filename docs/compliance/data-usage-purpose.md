# 利用目的明示

## 目的
PH-OS で取り扱う個人情報・要配慮個人情報・医療情報の利用目的を明示する。
患者、家族、連携先医療機関、介護事業者、薬局職員に対し、取得前または取得時に提示する文書として利用する。

## 取得する情報
- 患者基本情報: 氏名、生年月日、性別、住所、連絡先、保険情報
- 在宅訪問運用情報: 訪問予定、訪問記録、服薬状況、残薬情報、同意記録
- 処方・調剤関連情報: 処方箋原本スキャン、構造化処方、調剤実績、鑑査記録
- 連携情報: 報告書、照会記録、送付履歴、返信状況
- セキュリティ運用情報: 認証ログ、監査ログ、端末・IP・操作履歴

## 利用目的
1. 在宅患者訪問薬剤管理指導、居宅療養管理指導、服薬支援、残薬調整等の業務遂行
2. 処方内容確認、疑義照会、トレーシングレポート、主治医・ケアマネ・訪問看護との連携
3. 請求候補生成、算定根拠の保存、監査対応、個別指導・立入検査対応
4. 患者同意の範囲に基づく家族・施設・外部閲覧者への情報共有
5. システムの安全管理、なりすまし防止、不正アクセス検知、障害調査、再発防止
6. 法令・ガイドラインに基づく記録保存、報告、照会対応

## 第三者提供・委託
- 第三者提供は、法令に基づく場合または患者本人の同意範囲内で実施する。
- AWS などのクラウド基盤、メール送信事業者、監視基盤等を委託先として利用する場合がある。
- 委託先には秘密保持、安全管理措置、再委託管理を契約で義務付ける。

## 保存期間
- 薬歴、訪問記録、報告書、監査証跡は法令・ガイドラインに従い最低 5 年以上保存する。
- 認証・監査ログは 5 年保存を基準とし、アーカイブ方針に従って保管する。

## 開示・訂正・利用停止
- 本人または適法な代理人からの請求に対し、法令の範囲で開示・訂正・利用停止に対応する。
- 受付窓口、本人確認方法、回答期限はプライバシーポリシーおよび薬局内規程に従う。
