# ISMS キックオフチェックリスト

## 目的
CareViaX の ISMS 認証開始に向けて、技術側で揃っている統制と、経営 / 運用側で着手すべき項目を分離して管理する。

## 技術側で準備済みの項目
- アクセス制御ポリシー
- 変更管理 / リリース手順
- データ分類基準
- 脆弱性管理
- 委託先リスク評価
- セキュリティ教育計画
- 3省2ガイドライン統制マッピング
- 監査ログエクスポート
- バックアップ復旧試験 runbook

## キックオフ前に決める項目
- 審査機関候補 2 社以上の比較
- 初回見積の取得
- 対象組織 / 拠点 / システム境界の確定
- プロジェクトオーナー、情報資産管理責任者、事務局のアサイン
- 監査スケジュール候補日
- 予算上限と承認経路

## 初回ミーティングで確認する項目
1. 認証スコープ
2. 審査スケジュール
3. 不足文書と evidence の提出計画
4. 実地復旧試験の日程
5. 教育実施日と参加対象
